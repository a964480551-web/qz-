# 竖向板的原理介绍

竖向板的单位是发光强度，指的是光源在单位面积内发出的光通量

简单的说就是：面光的面积越大，灯光就越亮

所以优先调整面积，其次再调整亮度

球光的原理介绍

球光的单位也是是发光强度

简单的说就是：球光的体积越大光越强

暗

和竖向板的原理不同，球光遵循球的原理，最外的顶点光线最强，光强度往圆的两边衰减，适合打出光影感

球光的特点，中间直射的地方最亮一直到尽头，亮度往两边圆弧衰减

# 太阳

太阳高度角越低，阳光照射入房间内的范围越大。反之越少。

- 阳光在场景中的作用极为重要，是烘托场景氛围的最好工具，可以通过阳光来营造不同的氛围和光感，比如清晨、正午或者黄昏
- 阳光只有一个。这跟真实的自然环境道理相同。
- 阳光入射方向有窗，房间内就会有阳光。当窗口方向与阳光入射方向相反则无阳光效果。
- 指示线：仅代表阳光的入射方向。

# IES 光域网射灯

IES 光域网射灯，如名字就是模仿射灯的光束，增加场景层次，注意不要太亮太多影响已经打好的主光源

挑选合适的光束形状拖出使用即可，推荐亮度是 200、400，ies 的灯最好是融入场景，可以稍暗，但不要抢视觉中心。

# 自动灯带 / 灯线

柜体成品效果图灯带（非下单）我们做了自动适应长度。

灯线氛围模型灯带和虚拟灯带两种，区别在于第一种有厚度可见，第二种虚拟灯带：无厚度，可隐藏，可设置不可见，方向可视化

实体灯带常用场景：柜体

# 默认灯光一步出图

我们在默认灯光做了大量测试，其实直接就可以渲染了，当做好一个设计之后，可以尝试直接点默认灯光模版，直接渲染，大概率会有不错的效果，调整户外亮度就可以出图了，户外亮度参数推荐：10、25、40

# 如何正确利用灯光模版

每个灯光模版都有不同的效果，每个方案都不能百分百适用，只能达到一个相对稳定的效果，然后用这个作为底子，去微调一个适合当前方案的最佳灯光效果。

了解了原理，根据当前户型去增加或删减对应的灯光，就可以裂变出上百个灯光方案
