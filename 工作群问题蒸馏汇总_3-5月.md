## 一、渲染质量问题

### 1.1 曝光与过亮

| 问题描述 | 原因分析 | 解决方案 |
|---------|---------|---------|
| 户外/窗户曝光严重 | 外景球光过亮、聚光灯与面光重叠 | 换外景贴图；降低户外亮度；关闭聚光灯改用面光推光 |
| 榻榻米过亮 | 光线分布不合理 | 调整灯带亮度；增加面光补光平衡 |
| 餐厅区域曝光 | 窗外模型反光问题 | 降低窗外亮度；用挡光板遮挡 |
| 灯带亮一半 | 灯带尺寸与实际不符 | 进入渲染切换平面模式查看实际灯带长度，重新安装或手动放置虚拟灯带 |

**典型案例：**
- 餐厅曝光：将窗外亮度降低，或放置挡光板在窗户外
- 4K 渲染模糊：尝试解决接缝按钮；用单张竖向光板推光（亮度 0.几）补光

### 1.2 噪点与黑色区域

| 问题描述 | 原因分析 | 解决方案 |
|---------|---------|---------|
| 渲染出现噪点 | 灯光分布不均，局部过暗 | 添加面光平衡光线；给点面光补光 |
| 渲染出现黑色块/阴影 | 光线重叠、叠面、模型问题 | 删除重叠光源；检查自由建模叠面；更换模型 |
| 自由建模渲染黑线 | 墙体与生成面叠面 | 背景墙设计凸 1 毫；清除杂线 |
| 全景图某区域黑色 | 模型太老贴图丢失 | 重新拖取新模型；修复模型关联 |

### 1.3 颜色与材质异常

| 问题描述 | 原因分析 | 解决方案 |
|---------|---------|---------|
| 同一材质渲染两色 | 受光面不一致（凸凹不平） | 调整模型齐平；修改灯光角度 |
| 洞洞板渲染没洞 | 浅色材质覆盖贴图 | 换深色材质；月雾灰等浅色会覆盖洞洞纹理 |
| 玻璃太黑/失真 | 材质参数问题；清玻璃限制 | 调整材质参数；清玻璃只能同类更换，不能直接刷纹理玻璃 |
| 内圆弧造型颜色不一 | 自由建模叠面或光线计算问题 | 使用成品圆弧造型代替自由建模 |
| 地毯渲染粉色 | 地毯材质问题 | 重新渲染；更换地毯模型 |

## 二、灯光系统问题

### 2.1 灯光类型与使用

**虚拟灯带：**
- 位置：渲染模式 → 自定义灯光 → 拖出虚拟灯带
- 方向控制：看箭头方向（R 键拉伸长度宽度）
- 不亮原因：嵌入层板过深，统一往下移动 2-3 毫
- 双面发光：注意箭头向上，离地高度不要完全一致

**聚光灯 vs 面光 vs 球光：**
- 聚光灯：用于重点照明，但易造成曝光和重叠阴影
- 面光：推光柔和，推荐用于外景和补光
- 球光（大球）：用于室外环境光
- 球光（小球）：用于灯模型上方点缀

**筒灯/射灯问题：**
- 筒灯自发光无法完全关闭：进入渲染右侧面板关闭光源选项
- 嵌入式筒灯问题：部分模型需要挖槽，换平贴式筒灯
- 发光方向反了：剖面设计查看灯带箭头方向并移动

### 2.2 常见灯光故障

| 现象 | 诊断 | 处理 |
|-----|------|------|
| 灯不亮 | 其他地方太亮导致对比不明显 | 降低其他光源亮度；使用虚拟灯带 |
| 灯带透柜子 | 定制灯带生成虚拟灯带位置偏移 | 进入渲染调整虚拟灯带位置 |
| 神秘光源/光斑 | 自行上传模型带灯光；背景墙设计遗留 | 管理器搜索灯光全部删除；检查自由建模 |
| 发光背板不亮 | 定制兼容下单模型不发光 | 用素材库发光背板+材质刷 |

## 三、材质贴图问题

### 3.1 材质刷不上

| 场景 | 原因 | 解决办法 |
|-----|------|---------|
| 成品模型不能刷 | 标品限制 | 使用「软装消失」贴图刷消除多余部件；或重新拖模型覆盖 |
| 收藏后不能刷 | 收藏后变为独有素材，避免二次修改出错 | 刷之前收藏；或重新拖原模型 |
| 洞洞板刷后没洞 | 贴图被覆盖 | 不换材质直接换颜色板；或换其他颜色板 |
| 玻璃材质刷不上 | 清玻璃只能同类更换 | 直接拖模型玻璃覆盖；不能贴图刷 |
| 家配模型不能刷 | 家配标品限制 | 登记做定制门板或素材模型 |
| 台面纹理不能旋转 | 老版本可旋转新版本按钮缺失 | 重新生成台面；开发排期修复 |

### 3.2 贴图上传与处理

**个人上传流程：**
1. 贴图需 PS 处理四周加砖缝（如果是瓷砖）
2. 自建分类目录后上传
3. 上传失败点「解析失败」查看具体原因
4. SU 模型上传：转 Max 效果最佳；或材质统一后直传

**常见贴图问题：**
- 自己上传瓷砖默认白缝 → PS 改黑缝再传
- 连纹砖 → PS 先拼起来合成一张传
- 岩板/大理石没反射 → 侧角度才明显反光，正对无光源时弱
- 哑光贴图需求 → 可后台处理上传

## 四、模型相关问题

### 4.1 模型上传与登记

**登记入口：**
- 模型登记表：https://docs.qq.com/sheet/DVVhTQkVKb293WUR2?tab=BB08J2
- 找同款网站：https://3d.znzmo.com/

**上传注意事项：**
- 3DMax 归档尺寸超限时：先缩小到总宽 8000-9000，传后等比缩放再调回
- SU 上传失败：材质数量过多；最好转 Max 保存一次
- 上传后看不到：保存方案刷新；切换目录；检查解析状态
- 材质球上传：Max 里用 VRSense 导出器，选 VRSense 格式

### 4.2 模型搜索与使用

**搜索技巧：**
- 搜索不到时：返回主目录再搜；多搜几遍；检查有无空格
- 更有效方法：直接进入素材库目录手动滑动浏览
- 切换「最新」排序快速定位

**模型问题：**
- 模型渲染多出部件：删除原组合重新拖取
- 下架模型导致异常：保存刷新方案，删除下架模型重新拖取
- 模型破面/显示不全：修复显示面；上传时减面比例设为 0.5 或 1
- 床体渲染多出背景：模型问题，删除组合换新

### 4.3 特殊模型需求

**门窗类：**
- 弧形窗：素材库有圆弧窗，但不嵌入墙体；自由建模改造
- U 型窗：先建 U 型墙 → 改半墙 → 库里找 U 型窗组合
- 上下双窗：目前只能放一个，另一个背景墙开洞放不入墙门窗
- 转角推拉门（3+2）：用单扇拼合；参数化类型排期开发
- 门墙权限问题：未购买门墙无法使用参数化门窗，用素材库窗户代替

**吊顶/造型类：**
- 虎头线阳角：搜阳角；或编码 9349850511/2704576132 解组使用
- 圆弧造型：吊顶暂不支持复杂圆弧，需用成品模型拼接
- 壁龛：背景墙编辑开洞或拉伸实现
- 弧形墙落地窗：自由建模画框或垭口开洞

**软装类：**
- 并排放洗衣机+烘干机：编码 2754076023_0
- 岛台餐桌花模型：编码 7814125194
- 落地晾衣架：搜「衣帽架」
- 隐形门/特殊门：提供 CAD 定制制作

## 五、自由建模与硬装

### 5.1 自由建模限制

| 问题 | 现状 | 替代方案 |
|-----|------|---------|
| 不能倒圆弧边 | 技术限制 | 用成品垭口模型调整尺寸 |
| 转角窗不嵌入墙体 | 成品模型限制 | 参数化门窗（需门墙权限）或垭口+自由建模画框 |
| 踢脚线不识别圆弧 | 自动识别失效 | 自由建模单独拉弧形踢脚线 |
| 叠面问题 | 吸附不准或重复生成 | 凸 1 毫处理；清除吊顶/背景墙重试 |

### 5.2 户型与层高

- 单独空间调层高：不可行，需用吊顶补齐其他房间
- 跃层/别墅：分层画方案，全景图下载后合并到主方案漫游
- 户型门识别：自动识别入户门/室内门尚未实现
- 门互相替换：木门与钛镁合金门不能互换；入户门与室内门不能互换

## 六、全景图与漫游

### 6.1 全景图问题

| 问题 | 原因 | 解决 |
|-----|------|------|
| 渲染排队慢 | 任务并发显示延迟 | 实际已同时渲染；服务器陆续增加中 |
| 全景图切换视角没反应 | C 盘满了；浏览器问题 | 检查 C 盘空间；只开一个浏览器；保存刷新 |
| 两文件全景合并 | 需在一个链接中 | 主方案编辑漫游 → 添加本地图（下载其他方案全景） |
| 漫游点重置 | 修改房间名导致 | 保存后退出到设计界面再进；或改完名再保存一次 |

### 6.2 图册与分享

- 关闭分享方案：不给他人打开方案权限
- 下架效果图：在图册中操作
- 渲染好的案例图：部分未公开，需问同事索取或参考公开链接

## 七、高频操作指令

### 7.1 快速修复流程

**渲染异常排查清单：**
1. 保存方案 → 刷新浏览器重新进入
2. 检查 C 盘空间
3. 检查灯光重叠（聚光灯+面光）
4. 检查自由建模叠面
5. 检查模型是否下架/收藏后异常
6. 尝试「解决接缝」按钮

**灯光快速调整：**
- 太亮：降低亮度；关闭聚光灯；换面光
- 太暗：添加面光补光（亮度 0.4 左右）；增加球光
- 曝光：降低窗外亮度；用挡光板；换外景

## 八、待优化/待开发问题汇总

### 8.1 已登记排期

1. 截图识图搜索功能
2. 门墙与成品门互相替换
3. 吊顶添加更多线条样式（法式中古需求）
4. 转角窗参数化深度编辑
5. 筒灯模型默认不自发光
6. 铝扣板与浴霸对齐优化

8. 台面纹理旋转按钮权限修复
9. 搜索功能优化

### 8.2 技术限制说明

- 毛绒材质渲染：仅支持羊粒绒或模型碎片做法
- 水波纹灯带效果：需渲染器支持，暂用虚拟灯带渐变代替
- 发光背板定制下单：暂时不发光，效果图用素材库材质刷
- 油砂玻璃效果：渲染器版本读取不了光泽度，只能用凹凸贴图或降低透明度

## 九、服务流程规范

### 9.1 问题处理优先级

1. **紧急**：渲染黑图、方案打不开、客户等待中
2. **高优**：材质明显色差、模型重大缺陷、灯光系统性问题
3. **常规**：模型上传、贴图制作、功能咨询
4. **排期**：新功能需求、优化建议

### 9.2 用户沟通要点

- 询问方案号优先，便于精准定位
- 复杂问题远程查看（向日葵）
- 提供具体编码而非口头描述
- 登记制：模型/贴图需求走表格登记
- 个人上传优先引导至自助文档
